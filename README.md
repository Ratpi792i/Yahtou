# Yahtou
Yahtou is a from-scratch IEEE 802.1X authentication system built in Python.
It implements a RADIUS server with EAP-PEAP support, dynamic VLAN assignment,
tamper-proof audit logging (HMAC chaining), and a secured Flask admin portal.
A solution developed by students as part of a DevSecOps course
taught by Dr. Doudou FALL — ESP/UCAD, Master SSI 2025-2026.
